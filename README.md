# recur

write an entry. if a word you used has appeared before, recur surfaces that past entry. you can leave it as it is, or rewrite it. either way, your new entry is saved.

entries live at `~/.recur/entries.json`

---

## install

download the binary for your OS from [releases](../../releases):

- `recur-macos` — right-click → Open the first time
- `recur-windows.exe` — double-click
- `recur-linux` — `chmod +x recur-linux && ./recur-linux`

## or run from source

```bash
python3 recur.py
```
